# **App Name**: Safari Hotel Manager

## Core Features:

- Room Management: Track room availability, status (occupied, maintenance), and details.
- Reservation Management: Create, modify, and cancel reservations.
- Client Management: Maintain client profiles, history, and preferences to allow personalized customer service. Leverage Firestore to store customer data securely and efficiently.
- Automated Billing: Generate invoices, process payments, and issue receipts. Integrates with client profiles and reservation details.
- Real-time Dashboard: Provide at-a-glance views of key metrics such as occupancy rates, revenue, and upcoming reservations.
- Staff Management: Manage staff roles and permissions to ensure secure access and accountability.
- AI-Powered Suggestion Tool: Suggest optimized pricing based on historical data and current booking trends.

## Style Guidelines:

- Primary color: Deep teal (#008080) to evoke trust and professionalism.
- Background color: Light grey (#F0F0F0) for a clean and modern look.
- Accent color: Soft gold (#D4AF37) for highlighting key elements and calls to action.
- Headline font: 'Playfair', serif, for headlines or small amounts of text; Body font: 'PT Sans', sans-serif, for body text.
- Use clean, line-based icons for intuitive navigation. Icons should be monochromatic to maintain a minimalist design.
- Employ a grid-based layout with ample whitespace to create a clear and uncluttered interface.
- Use subtle transitions and animations to enhance user experience, such as loading indicators and smooth page transitions. Optimize animations for low-bandwidth environments.
